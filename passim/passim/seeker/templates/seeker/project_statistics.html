Notes:

No items are allowed to be without a project assigned to them. Should an item turn out to be without a project (when you delete project [{{project.name}}] from that item, the project cannot be deleted from that item. The proper way to 'clean' a whole project is:
  • Review all the items belonging to a particular project (manuscripts, sermons, authority files, historical collections)
  • Choose:
    • Add the proper (alternative) project to that item, so that it won't be 'project-orphaned'
    • Or remove the item (if you have the right), so that it won't have a chance to be 'project-orphaned'
  • Only then is it possible to remove the project association from all these items
