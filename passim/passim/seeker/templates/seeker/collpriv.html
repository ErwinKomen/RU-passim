Add items to this personal dataset

This personal dataset can be used to store a collection of Manuscripts, Sermon Manifestations, Sermons Gold or Super Sermons Gold. Adding such items can be done in this way:

  • Go to the listview of the items
  • Use the search filters to create a basket that contains the right items
  • Use the basket button "add to a dataset..." to add the items to the dataset you have created

Here are shortcuts to the four listviews:

Manuscripts: Goto manuscripts
 
Sermon Manifestations: Goto sermons
 
Sermons Gold: Goto gold sermons
 
Super Sermons Gold: Goto super sermons gold
