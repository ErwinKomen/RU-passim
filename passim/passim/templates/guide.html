Short user manual

Home page

The home page of the Passim web application comes up with three pie charts that give a bird's eye view of the current status. This status is provided for three categories:

  • Manifestations
  • Authority files (AFs)
  • Manuscripts

Each of the pie charts show a red, orange and a green area. These area's signal progress in processing the manifestations, AFs and manuscripts. Each color stands for one area of progress:
red Initial items have been imported (often automatically), but no processing has begun yet.
orange Edited items are being edited and reviewed by the PASSIM team.
green Approved items have been manually approved.

AF histograms

The Authority File (AF) listview provides a 'histogram' button. The histogram shows how the total amount of AFs is divided in terms of 'manifestation frequency'. Each AF has such a frequency, and this amounts to the number of manifestations that are linked with the AF.

The AF histogram adapts itself to the current set of filtered AFs.
